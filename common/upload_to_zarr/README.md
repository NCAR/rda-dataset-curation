
# Uploading a dataset to zarr

### Create A config
```
usage: ./createDatasetConfig.py [-h] [--json] [--yaml] [--skip_errors]
                                [--title TITLE]
                                files [files ...]
```


### Upload Files
```
usage: ./upload_zarr.py [-h] [--config CONFIG] [--skip_errors] [--title TITLE]
                        files [files ...]
```
